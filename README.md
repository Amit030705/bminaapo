# BMI Calculator

A modern and responsive **BMI Calculator** built using React and bundled with Parcel. This app helps users calculate their **Body Mass Index (BMI)** using height and weight inputs with instant results.

## 🚀 Live Demo

Add your deployed link here:
`bminaapo.netlify.app`

## 📌 Features

* Instant BMI calculation
* Clean and responsive UI
* Real-time validation
* BMI health categories:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* Fast development with Parcel
* Mobile friendly design

## 🛠️ Tech Stack

* React
* Parcel
* JavaScript (ES6+)
* HTML5
* CSS3

## 📊 BMI Formula

```text id="7z7ltw"
BMI = Weight (kg) / Height² (m²)
```

## 📂 Installation & Setup

Clone the repository:

```bash id="1hjk8v"
git clone https://github.com/your-username/bmi-calculator.git
```

Move into project folder:

```bash id="2v98qa"
cd bmi-calculator
```

Install dependencies:

```bash id="6q0fne"
npm install
```

Run development server using Parcel:

```bash id="6cp1jv"
npx parcel index.html
```

Build for production:

```bash id="g2tmzn"
npx parcel build index.html
```
